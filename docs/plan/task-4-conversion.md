## All Converters Must

1. **Preserve original type labels** (View A). View B generated by `convert_all.py`.
2. **Build `query_types`** with positive + 2-5 frequency-weighted negatives (holdout types excluded).
3. **Validate spans**: `text[start:end] == entity["surface"]`. Log and skip broken spans.

### `lib/span_computer.py` (occurrence-based, NOT sequential) {#span-computer}

Used for LLM output in [Task 6](task-6-labeling.md). Never trust LLM-provided character offsets.

```python
import re
from collections import defaultdict
import unicodedata

def normalize(text: str) -> str:
    return unicodedata.normalize("NFKC", text)

def compute_all_spans(text: str, entities: list[dict]) -> list[dict]:
    """Occurrence-based matching. Does not assume entity order."""
    # Precompute all occurrences for each unique surface
    occurrence_map: dict[str, list[tuple[int, int]]] = {}
    for ent in entities:
        surface = ent["surface"]
        if surface not in occurrence_map:
            occurrence_map[surface] = [
                (m.start(), m.end())
                for m in re.finditer(re.escape(surface), text)
            ]
            # NFKC fallback for CJK full-width/half-width variants
            if not occurrence_map[surface]:
                norm_text = normalize(text)
                norm_surface = normalize(surface)
                norm_matches = [(m.start(), m.end())
                    for m in re.finditer(re.escape(norm_surface), norm_text)]
                if norm_matches:
                    # Only keep if original text at same offsets matches
                    occurrence_map[surface] = [
                        (s, e) for s, e in norm_matches
                        if text[s:e] == surface
                    ]

    # Track consumed occurrences
    used: dict[str, int] = defaultdict(int)
    result = []

    # Sort by first possible occurrence for text-order assignment
    def first_occ(ent):
        occs = occurrence_map.get(ent["surface"], [])
        return occs[0][0] if occs else float('inf')

    for ent in sorted(entities, key=first_occ):
        surface = ent["surface"]
        occs = occurrence_map.get(surface, [])
        idx = used[surface]
        if idx < len(occs):
            ent["start"], ent["end"] = occs[idx]
            used[surface] = idx + 1
            result.append(ent)
        else:
            log_missing(text, ent)
    return result
```

For conversation format: same logic scoped to `turns[turn_index].text`.

Adversarial tests required: repeated substrings, overlapping entities, CJK mixed with Latin, out-of-order entities, full-width/half-width variants.

### `lib/bio_to_spans.py` — includes detokenization {#detokenization}

```python
def detokenize(tokens: list[str], language: str) -> str:
    if is_cjk_language(language):
        return smart_cjk_join(tokens)  # No spaces between CJK, spaces around Latin
    else:
        text = " ".join(tokens)
        text = re.sub(r'\s+([.,!?;:)\]}\%])', r'\1', text)  # No space before punct
        text = re.sub(r'([\(\[\{"])\s+', r'\1', text)        # No space after open
        return text
```

**Critical**: If dataset provides raw/original text alongside tokens, ALWAYS prefer raw text. Reconstruct from tokens only as fallback.

## Per-Dataset Conversion

### B2NERD (hardest)

- EN with `pos` field: Use `pos` directly. Verify inclusive vs exclusive end on 10 samples first (see [Pre-Task 4](task-4-pretask.md)). Add +1 if inclusive.
- ZH: If Pre-Task 4 confirms `pos` exists, use it. Otherwise: `str.find()` for first-match, log ambiguous cases to `data/processed/b2nerd_ambiguous.jsonl`.
- EN without `pos`: Word-boundary regex (`\b{name}\b`).
- Data is in `B2NERD_data.zip` — extract first.

### BIO Datasets (open-ner, klue, multiconer)

- Reconstruct char offsets from token-level BIO tags.
- Use `detokenize()` for text reconstruction (not naive space join).
- open-ner CJK: Detect tokenization style. Multi-char CJK tokens → join WITHOUT spaces.
- klue: Individual characters → direct mapping.
- multiconer: CoNLL format, character-per-line for CJK.

### fiNERweb

- Native `{start, end, label}` spans — validate and emit.
- Language from HF dataset config/subset name (NOT langdetect).
- **Per-language cap: 50K** (EN/JA/ZH/KO). Stratified random selection.

### chinese_ner_sft

- Native `start_idx`/`end_idx`. Some may be null → fall back to string matching.

## Cross-Dataset Deduplication (`lib/dedup.py`) {#dedup}

**Merge-on-duplicate**, not drop-on-duplicate:

- open-ner-standardized (60 types) and open-ner-core-types (3 types) share underlying texts. Dropping one loses supervision signal.
- Same text hash from same dataset family: merge entities (union), merge query_types, keep provenance list, set confidence to max.
- Same text hash from different families: merge entities (union). Same span + different type → keep both.
- Only DROP true duplicates (same text, same entities, same types).
- Near-duplicate: MinHash with Jaccard 0.85 on character 5-grams (optional).
- Log dedup stats.

## Train/Val Split (`lib/splitter.py`)

Stratified 95/5 by `language × source`. Applied after dedup.

### Zero-Shot Split Handling

Do NOT move entire examples to `zero_shot_eval`. Instead:
- Strip held-out-type entities from examples, keeping the rest in train.
- Only create dedicated `zero_shot_eval` examples for texts where ALL entities are held-out types.
- This preserves supervision on non-held-out types that co-occur.

## View B Generation (`convert_all.py`)

After all converters emit View A:
1. Load `type_mapping.json` from [Task 3](task-3-mapping.md).
2. For each example with mappable types, generate a View B copy.
3. Remap `type` → canonical label, keep `original_type`.
4. Suffix source with `_canonical`.
5. Remap `query_types` to canonical strings.
6. Both View A and View B go into `train.jsonl`.
